Quick commit and push current changes.

1. Run `git status` and `git diff --stat` to see what changed
2. Draft a concise commit message based on the changes
3. Stage relevant files (not .env, credentials, or large binaries)
4. Commit with the drafted message
5. Push to the current branch

Ask for confirmation before pushing if the branch is main/master.
